== Description ==

Enables you to add images for gallery to appropriate pages...Using the short code [wp-slides] to displayed the images into the pages or post. Use this short code into php file <?php echo do_shortcode( '[wp-slides]' );?>.

Images alt text and images name can be modified.

Search options to find particular page images.   

== Installation ==

1. Upload the `SNS Image Gallery` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
